# 🌐 Universal Document Image Translator to PDF

This app extracts text from images (like Arabic scans), translates it into English, and builds a downloadable PDF.

## 🛠️ Tech Stack
- Frontend: Streamlit
- OCR: EasyOCR
- Translation: Translators (Google)
- PDF Generation: FPDF2

## 🚀 Run Locally
1. Install Dependencies: `pip install -r requirements.txt`
2. Launch App: `streamlit run app.py`
